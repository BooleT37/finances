# Sources

A source is an optional transaction property. Sources are used exclusively in the transactions context — they are never referenced in other features.

## Transaction Entry Aid

The primary purpose of a source is to simplify adding new transactions. When entering a new transaction, it's important to know which transactions have already been recorded for that source. To help with this, the transaction form shows a hint next to the source field displaying the most recent transactions for the selected source. This allows you to find where you left off in the banking app.

## Transactions Table

Sources are shown as a column in the transactions table and can be used to filter transactions.

## Import

Each source can be assigned its own statement parser for bulk-importing transactions

Sources can be reordered in settings. The order is reflected everywhere the list of sources is displayed.
