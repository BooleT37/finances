# Finances

A personal finances tool for logging income and expenses from different sources, budgeting, tracking savings and subscriptions, and viewing statistics.

## Tech Stack

- **Framework**: [TanStack Start](https://tanstack.com/start/latest) (full-stack React)
- **Deployment**: Cloudflare Workers
- **API**: tRPC
- **ORM**: Prisma
- **Database**: PostgreSQL
